Question 1 – 1000 words, 5 points

For the Take Home Exam of Digital Media Technology, you need to carry out a small research project in which you to attempt answer a research question using a raw data set which can be accessed online. The data set needs to be stored as a relational database.

During the research project you need to move through the following phases:

  • Database design

Select a raw data set and formulate a research question. Next, develop an Entity-Relationship Diagram for the data set.

  • Database implementation

Using the ERD as a basis, define a number of SQL CREATE statements which can implement the various entities which you had defined as tables.

  • Data entry

Obtain the data  from an online source and reorganise them in such a way that they can be entered into the database. Data is generally made available in the form of XML. Using XSLT stylesheets, convert the XML data into SQL INSERT statements.

  • Data analysis

Use SQL to query the database and to produce visualisations which may help you to answer you central research question.

In your research project, you may focus on the three raw datasets which are described on the following page: https://digitalscholarship.nl/DMT/Assignments/ResearchProject.html
It is also allowed to work with a data set which is not in the list. Some inspiration can be found on the following websites: 

  • Collection of data sets developed by Miriam Posner
  • Quantitative Analysis: Humanities Datasets
  • DH Toychest developed by Alan Liu

Write a brief essay in which you discuss your approach and in which you discuss the main findings of your research project.

 

Question 2 – 5000 words, 2 points

The Walt Whitman Archive is an online project, hosted by the University of Nebraska-Lincoln, which offers access primary and secondary sources about the American poet, essayist and writer Walt Whitman (1819-1892). The archive demonstrates applications of  the technologies you have been taught about during DMT, as well as others you may not be familiar with. Explore the project, and answer the following questions in a running text.
  • Which of the functionalities that you see on the website have (likely) been implemented using which technologies that have been discussed during DMT? Can you think of additional functionalities that could be implemented with these technologies?
  • Which of the project’s functionalities go beyond the affordances of the technologies featured in DMT?
  • Which features of the project do you find particularly striking? Why?
  • Also analyse the rationale for creating the Walt Whitman Archive. Which new possibilities for research or access to source materials does it open up? For whom is the site intended?
